# 🚀 亚马逊关键词埋词检查工具

一个强大的在线工具，专为亚马逊卖家设计，用于检查产品文案中的关键词覆盖情况，助力提升产品搜索排名。

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/yourusername/amazon-keyword-checker/blob/main/LICENSE)
[![GitHub Pages](https://img.shields.io/badge/demo-online-green.svg)](https://yourusername.github.io/amazon-keyword-checker/)

## ✨ 功能特点

### 核心功能
- 🔍 **智能关键词检查** - 快速检测文案中的关键词覆盖情况
- 🌍 **多语言支持** - 支持英语、德语、法语、意大利语、西班牙语等多种语言的单复数匹配
- ✨ **可视化高亮** - 匹配的关键词在文案中高亮显示
- 📊 **详细统计** - 实时显示匹配率、已匹配/未匹配数量
- 📋 **一键操作** - 支持去重、复制、重置等快捷功能

### 特色功能
- 🎯 实时检测并提示重复关键词
- 🚫 自动过滤中文字符和标点符号
- 📱 完美适配移动端设备
- 🎨 现代化UI设计，流畅的动画效果
- ⚡ 纯前端实现，无需服务器支持

## 📖 使用说明

### 1.输入关键词

- 在左侧输入框输入需要检查的关键词
- 支持换行分隔或逗号分隔
- 系统自动过滤中文和标点符号

### 2.粘贴产品文案

- 在右侧输入框粘贴产品文案
- 支持产品标题、五点描述、详情等内容

### 3.执行检查

- 点击"检查关键词"按钮
- 查看高亮显示的匹配结果
- 查看详细的统计数据

### 4.导出结果

- 点击统计卡片查看详细列表
- 一键复制未匹配关键词
- 便于后续优化文案
